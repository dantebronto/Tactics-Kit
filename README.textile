h1. Tacktics Kit: A tactical RPG

Tacktics Kit is a proof of concept game using only jQuery and HTML. For ease of use, it also uses Sinatra's built-in web server.

h1. Setup

p. First clone the project. Then use bundler to install the necessary dependencies:

<pre>bundle install</pre>

p. Then you should be able to run:
<pre>ruby server.rb</pre>

p. From here you can simply hit http://localhost:4567/ to begin playing. Enjoy!
